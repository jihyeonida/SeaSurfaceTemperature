# HYCOM 수온 데이터셋 (수심별 단일 CSV)

## 개요 (HYCOM, https://www.hycom.org)
- 내용: HYCOM 실험 결과를 수심별로 분리한 단일 CSV 파일 모음
- 기간: 2017-01-01 00:00:00 ~ 2020-01-31 21:00:00  
- 좌표계: 위도/경도 (decimal degrees), 저장 시 소수 5자리 반올림
- 단위: `water_temp` = °C
- 결측/중복: 최종 파일 기준 NaN 0, 중복 S
- 파일명 규칙: `depth_{수심}.csv`  
  예) `depth_150p0.csv` → 150.0 m

## 컬럼 설명
모든 파일 공통 컬럼:
- `time` (UTC, `YYYY-MM-DD HH:MM:SS`)
- `lat`, `lon` (deg; 저장 시 소수 5자리 반올림)
- `depth` (m; 각 파일은 단일 수심만 포함)
- `water_temp` (°C)

## 원자료 및 격자
- 사용한 실험과 기간/격자:
  - `expt_56.3` (2014-10-01 00:00 ~ 2016-05-01 09:00) — **0.08×0.08**
  - `expt_57.2` (2016-05-01 12:00 ~ 2017-02-01 09:00) — **0.08×0.08**
  - `expt_92.8` (2017-02-01 12:00 ~ 2017-06-01 09:00) — **0.08×0.08**
  - `expt_57.7` (2017-06-01 12:00 ~ 2017-10-01 09:00) — **0.08×0.08**
  - `expt_92.9` (2017-10-01 12:00 ~ 2018-01-01 09:00) — **0.08×0.08**
  - `expt_93.0v` (2018-01-01 12:00 ~ 2018-12-04 09:00) — **0.08×0.08**
  - `expt_93.0y` (2018-12-04 12:00 ~ 2020-12-31 21:00) — **0.04×0.08**  
- 격자 전환 경계 시점: 2018-12-04 12:00:00 (93.0v → 93.0y, 위도 해상도 0.08→0.04)  
- 경도 증분은 명목상 0.08°이며, 반올림 영향으로 약 0.07995~0.08008° 범위의 미세 편차가 존재할 수 있음.

## 생성/병합 규칙
- 보간/재샘플링/리그리드: 수행하지 않음 (원좌표/원주기 유지)
- 중간 정리: `expt_93.0v` 원본 내부에서 같은 `(time,lat,lon)` 키 중복/충돌을 유효값(Non-NaN) 우선 + 뒤에 등장한 행 우선으로 1개만 유지
- 실험 간 병합: 같은 `(time,lat,lon,depth)`가 여러 실험에 존재하면  
  1) 유효값(Non-NaN) 우선
  2) 최신 실험(목록 뒤쪽) 우선으로 1개 선택
- 저장 전 처리:
  - `lat, lon` 소수 5자리 반올림
  - `time/lat/lon/depth` 결측 행 제거
  - `water_temp` NaN 행 제거
- 최종 품질 확인:
  - 파일별 `(time,lat,lon)` 중복 0
  - 전체 `(time,lat,lon,depth)` 중복 0, NaN 0
